Basic Essential Concepts (Foundation)

  • Header Files: #include <stdio.h>, <math.h>, <string.h>, <ctype.h> etc.
  • Main Function: Entry point of a C program.
  • printf/scanf: Used for formatted input/output.
  • Data Types: int, float, char, double, void.
  • Storage Size: int(2/4 bytes), float(4 bytes), double(8 bytes).
  • Format Specifiers: %d (int), %f (float), %c (char), %s (string), %lf (double).
  • Type Modifiers: short, long, signed, unsigned.
  • Escape Sequences: \n (newline), \t (tab), \r (carriage return), \\ (backslash).
  • Operators: Arithmetic (+,-,*,/,%), Relational (<,>,<=), Logical (&&,||,!), Bitwise (& | ^ ~ << >>).
  • Conditionals: if, else-if, else.
  • Loops: for, while, do-while.
  • Functions: returnType functionName(parameters);
  • Arrays: int a[10]; → continuous memory.
  • Strings: char s[20]; → ends with '\0'.
  • Pointers: int *p; → stores address of variable.
  • DMA: malloc, calloc, realloc, free.
  • Files: fopen, fclose, fgetc, fputc, fgets, fputs.

CONCISE DEFINITIONS — QUICK REVISION

  • Variable: Named memory location used to store data.
  • Constant: Fixed value that cannot change during execution.
  • Keyword: Reserved word having special meaning in C.
  • Operator: Symbol that performs arithmetic or logical operation.
  • Compiler: Converts C code to machine language.
  • Function: Block of reusable statements.
  • Array: Collection of similar data items stored in contiguous memory.
  • String: Sequence of characters ending with '\0'.
  • Pointer: Variable that stores address of another variable.
  • Dynamic Memory: Allocation during runtime using malloc/calloc.
  • Recursion: Function calling itself.
  • Call by Value: Pass copy of variable.
  • Call by Reference: Pass address of variable.
  • File: Data stored permanently on disk.
